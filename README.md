# REST API
 - API: '/vcambio' (post), para crear un nuevo registro
 - API: '/vcambio' (get),  obtener todos los registros de la coleccion
 - API: '/vcambio/<id>' (get), obtener por id
 - API: '/vcambio_q/<q>' (get), obtener por busqueda
